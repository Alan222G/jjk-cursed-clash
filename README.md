# Jujutsu Kaisen: Cursed Clash

2D competitive fighting game built with **Phaser 3** and **Vite**.

## Features
- **Deterministic Combat Engine**: Custom state machine for frame-perfect fighting mechanics.
- **Cursed Energy System**: Resource management for special abilities.
- **Domain Expansion**: Dramatic environmental changes and "Sure-Hit" mechanics.
- **Domain Clash QTE**: Rhythm-based minigame for domain tug-of-war.
- **Characters**: 
  - **Gojo Satoru**: Control over space (Blue/Red/Unlimited Void).
  - **Ryomen Sukuna**: King of Curses (Cleave/Dismantle/Malevolent Shrine).

## Controls
### Player 1 (Left)
- **WASD**: Movement / Jump / Duck
- **J**: Light Attack
- **K**: Medium Attack
- **L**: Heavy Attack
- **U**: Special Ability (Uses Cursed Energy)
- **I**: Domain Expansion (100% CE)
- **Shift**: Block

### Player 2 (Right)
- **Arrows**: Movement / Jump / Duck
- **Numpad 1**: Light Attack
- **Numpad 2**: Medium Attack
- **Numpad 3**: Heavy Attack
- **Numpad 4**: Special Ability
- **Numpad 5**: Domain Expansion
- **Numpad 0**: Block
